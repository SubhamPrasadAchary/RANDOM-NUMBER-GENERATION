Key Features
Random Number Generation: Generates numbers from 0 to 36 at the click of a button

Visual Feedback:

Green for 0 ,
Red for even numbers ,
Black for odd numbers

User Experience:

Loading state during number generation
Clear visual indicators for different number types
Responsive design for all screen sizes

Frontend: React.js

State Management: React Hooks (useState)

Styling: CSS (with class-based styling)

Build Tool: Create React App
